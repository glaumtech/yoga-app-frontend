# Banner Images

Place your banner images in this directory.

## Required Images

- `banner1.jpg` - First banner image
- `banner2.jpg` - Second banner image  
- `banner3.jpg` - Third banner image

## Image Specifications

- **Format**: JPG or PNG
- **Recommended Size**: 1200x400 pixels (3:1 aspect ratio)
- **File Size**: Keep under 500KB for optimal performance

## Adding More Banners

To add more banners, update the `bannerImages` list in `lib/presentation/screens/home/home_screen.dart`:

```dart
final bannerImages = [
  'assets/images/banners/banner1.jpg',
  'assets/images/banners/banner2.jpg',
  'assets/images/banners/banner3.jpg',
  'assets/images/banners/banner4.jpg', // Add more as needed
];
```
